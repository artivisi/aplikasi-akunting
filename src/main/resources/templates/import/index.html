Import Data Lengkap

Upload file ZIP hasil ekspor untuk mengganti seluruh data

Peringatan

Import akan menghapus SELURUH data yang ada dan menggantinya dengan data dari file. Pastikan Anda sudah mengekspor data terlebih dahulu sebagai backup.

Petunjuk

Format File

File ZIP yang dihasilkan dari menu Ekspor Data. File berisi CSV untuk semua entitas dan folder dokumen.

Proses Import

  1. Semua tabel akan dikosongkan (truncate)
  2. Data dari CSV diimpor sesuai urutan dependensi
  3. File dokumen dikembalikan ke folder storage

Data yang Diimpor

Pengaturan perusahaan, bagan akun, template jurnal, klien, proyek, karyawan, transaksi, jurnal, dokumen, dan data lainnya.
