# Release 2025.12.6

## Highlights

Security hardening release focused on Content Security Policy (CSP) improvements, eliminating `unsafe-inline` and `unsafe-eval` directives for enhanced XSS protection.

## What's New

### Security Hardening

- **CSP Nonce-based Script Protection**: Added nonce attributes to all 50+ inline scripts across templates
- **Alpine.js CSP Build**: Upgraded to Alpine.js 3.15.3 with CSP-compatible expression parser
- **Style CSP Compliance**: Eliminated `unsafe-inline` for styles using hash-based allowlisting

## Improvements

- Upgraded HTMX from 2.0.4 to 2.0.7
- Upgraded Alpine.js from 3.14.8 to 3.15.3 (CSP build)
- Fixed template search dropdown visibility with pure JavaScript handlers
- Fixed sidebar Chart of Accounts link using correct ACCOUNT_VIEW authority
- Added HTMX indicator style hash to CSP policy
- Fixed QuickTransactionTest timing issue with proper navigation wait

## Bug Fixes

- Fixed template search dropdown not appearing when typing in search box
- Fixed Chart of Accounts link missing from sidebar navigation
