# Release 2026.03.2

## Highlights

Security hardening and code quality release. Fixes mass assignment vulnerabilities (S4684), resolves all Dependabot security alerts, and addresses 60+ SonarCloud code smells.

## What's New

### Security

- Replace JPA entities with form DTOs in all 19 MVC controllers (S4684 mass assignment fix)
- Fix @Transactional self-invocation bypass in TaxTransactionDetailService (S6809)
- Resolve 6 Dependabot security alerts (jackson-core CVEs, removed Pulumi)

### Code Quality

- Extract 30+ duplicated string literals into constants across controllers and services (S1192)
- Remove generic wildcard types in TaxExportApiController (S1452)
- Fix form label-input associations for accessibility in tax detail and invoice/bill templates (Web:S6853)
- Fix low-contrast text in client/vendor print templates (css:S7924)
- Remove unused toForm() methods in BillController and InvoiceController (S1144)
- Suppress BEAN_PROPERTY_INJECTION false positives in SpotBugs

### Infrastructure

- Remove Pulumi deployment (no longer used), resolves tar + minimatch CVEs
- Override jackson-bom 3.1.0 and jackson-2-bom 2.21.1 for CVE fixes

## Improvements

- Fix Thymeleaf Layout Dialect 4.0.0 (Groovy 5.0.4 NPE in DecorateProcessor)
- Alpine.js CSP-compatible form components for bills, invoices, recurring transactions

## Bug Fixes

- Fix Groovy 5.0.4 NPE breaking Thymeleaf layout rendering in production
- Fix Alpine.js line items not rendering in CSP build (bills, invoices, recurring forms)

## Breaking Changes

- None

## Migration Guide

No migration steps required. Flyway V003 checksum update may be needed on servers with older schema history (auto-handled by deploy script).

## Dependencies

- Spring Boot 4.0.3
- Java 25
- Thymeleaf Layout Dialect 4.0.0
- Jackson BOM 3.1.0 / Jackson 2 BOM 2.21.1
- SpotBugs 4.9.8.2 with FindSecBugs 1.13.0
