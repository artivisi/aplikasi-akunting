# Release 2026.02.2-RELEASE

## Highlights

- **Transaction API**: REST endpoints for listing, viewing, and posting transactions (draft creation with approval workflow)
- **Data Import API**: Programmatic data import endpoint for AI-driven client onboarding
- **Per-Client Theming**: Ops-driven theming system via `application.properties` — each client gets their own logo, colors, and footer text
- **Device Session Management**: UI for viewing and revoking device sessions from user detail page

## What's New

### Transaction API (Phase 7)
- `GET /api/transactions` — list transactions with pagination and date range filtering
- `GET /api/transactions/{id}` — transaction detail with journal lines
- `POST /api/transactions` — create draft transactions (requires approval via web UI)
- Bearer token authentication, `transaction:read` and `transaction:write` scopes

### Data Import API
- `POST /api/data-import` — upload CSV seed packs programmatically
- Supports all import types: COA, templates, products, BOMs, company config, transactions
- Enables AI-driven client onboarding workflows

### Per-Client Theming
- `ThemeConfig` reads `app.theme.name`, `app.theme.footer-text`, `app.theme.dir` from properties
- `ThemeInterceptor` injects theme data into all Thymeleaf views
- Theme CSS overrides `:root` custom properties — all Tailwind utility classes pick up new colors automatically
- Theme assets served from external filesystem directory (not bundled in JAR)
- Adding a new client requires only deploy config changes, no code rebuild

### Device Session Management UI
- View active device sessions on user detail page
- Revoke individual sessions

## Bug Fixes

- Fixed CVE in tar dependency: upgraded override to >=7.5.8 to resolve hardlink/symlink escape vulnerability

## Migration Guide

No schema migration required. Standard upgrade process:

1. Pull latest code
2. Run `./mvnw clean package`
3. Deploy using Ansible or remote-build script

For theming, add to `application.properties`:
```properties
app.theme.name=artivisi
app.theme.footer-text=ArtiVisi Intermedia
app.theme.dir=/opt/aplikasi-akunting/themes
```
