# Release 2026.02-RELEASE

## Highlights

- **Bank Reconciliation (Phase 9)**: CSV import, 3-pass auto-matching, manual match, reconciliation reports
- **AI Analysis Reports**: Structured report publishing with per-industry KPIs for 4 industries
- **Financial Analysis API**: Read-only API endpoints for AI tools to access company financials
- **License Change**: Migrated from AGPL-3.0 to Apache License 2.0
- **Code Quality**: 100+ SonarCloud issues resolved, test coverage improved to 75%+

## What's New

### Bank Reconciliation (Phase 9)
- Bank statement import with CSV parsing (BCA, Mandiri, BNI, BSI, CIMB)
- Custom parser configuration for unsupported banks
- Auto-matching: 3-pass algorithm (Exact, Fuzzy Date, Keyword)
- Manual match, Bank-Only, Book-Only classification
- Reconciliation reports (print/export)
- 5 permissions: VIEW, IMPORT, MATCH, COMPLETE, CONFIG
- 16 Playwright functional tests

### AI Analysis Reports
- Publish structured reports via POST /api/analysis/reports (JSONB: metrics, findings, recommendations, risks)
- Per-industry KPIs: IT Service, Online Seller, Coffee Shop, Campus
- Web UI: list + detail pages with industry badges
- ANALYSIS_REPORT_VIEW permission, analysis:write scope
- 4 industry-specific Playwright functional tests

### Financial Analysis API
- GET /api/analysis/company - company info
- GET /api/analysis/snapshot - monthly financial snapshot
- GET /api/analysis/income-statement - revenue/expense breakdown
- GET /api/analysis/receivables - outstanding receivables
- OAuth 2.0 Device Flow authentication for AI tools

### AI-Assisted Transactions
- OAuth 2.0 Device Flow for AI tool authentication
- Template metadata API for AI template matching
- Direct transaction posting (skip draft)

## Migration Guide

No schema migration required. Standard upgrade process:

1. Pull latest code
2. Run `./mvnw clean package`
3. Deploy using Ansible: `ansible-playbook -i inventory.ini deploy.yml`
4. Flyway will apply V006 (bank reconciliation schema) and V007 (bank reconciliation seed data) automatically
