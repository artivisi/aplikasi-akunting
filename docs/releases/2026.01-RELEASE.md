# Release 2026.01-RELEASE

## Highlights

- **Consolidated CI/CD Pipeline**: Single workflow with parallel jobs for faster feedback
- **DAST Always-On**: Security scanning runs with every test, not just in CI
- **Spring Boot 4.0.1**: Updated to latest Spring Boot version
- **PostgreSQL 18**: Infrastructure upgraded to PostgreSQL 18 with optimized settings

## What's New

### CI/CD Improvements
- Consolidated 4 separate workflows (ci, security, codeql, dast) into single `ci.yml`
- 6 parallel jobs: secret-scan, build-and-test, spotbugs, codeql, semgrep, dependency-check
- JaCoCo coverage summary displayed in GitHub Actions job summary
- Codecov integration for coverage tracking and badges
- DAST runs with every `./mvnw test` (~3 minutes overhead)

### Security Enhancements
- Graybox DAST testing with endpoint enumeration from controller mappings
- Improved CSP header coverage for all dispatcher types
- Fixed ZAP report generation for stable cross-version compatibility

### Infrastructure
- PostgreSQL 18 upgrade with optimized settings for 2GB VPS
- JVM G1GC tuning for small heap sizes
- Capacity planning documentation

## Improvements

- Simplified test commands - no more `-Ddast.enabled=true` flag needed
- Maven wrapper version corrected to 3.9.9
- Updated ZAP client API to 1.17.0 for ZAP 2.17.0 compatibility
- Removed obsolete session summary files

## Bug Fixes

- Fixed DAST report generation bug (Reports.generate API compatibility)
- Fixed SpotBugs false positive for error controller CSRF
- Resolved deployment issues found during production upgrade
- Addressed DAST findings and improved CSP header coverage

## Breaking Changes

- None

## Migration Guide

No migration required. Standard upgrade process:

1. Pull latest code
2. Run `./mvnw clean package`
3. Deploy using Ansible: `ansible-playbook -i inventory.ini deploy.yml`
