## Patch Release 2025.12.2

Improvements to transaction form UX, seed data, and test coverage.

### What's New

- **Initial Balance Template**: New seed data template for recording initial account balances
- **Flexible Bank Templates**: Additional banking transaction templates with configurable accounts

### Improvements

- **Consolidated Transaction Form**: Merged template execute functionality into main transaction form
  - Streamlined workflow for creating transactions from templates
  - Single unified form for all transaction types

- **Dashboard UI**: Improved spacing between KPI cards and content sections

- **Journal Preview**: Enhanced account code styling in preview mode

- **Tax Expense Accounts**: Restructured tax expense account hierarchy in seed data for better categorization

- **Template Types**: Changed multi-variable templates to DETAILED type for correct behavior

### Bug Fixes

- Fixed transaction form preview display issues
- Fixed sequence number generation in transaction form
- Updated tests to align with consolidated transaction form

### Documentation

- Added delayed tax payment scenarios to user manual (PPh and PPN)

### Testing

- Added integration tests for PayrollService#postPayroll

### Technical Details
