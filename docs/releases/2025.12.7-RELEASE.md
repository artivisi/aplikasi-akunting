## Highlights

This release adds tax profile management, template versioning, and several UI improvements for better transaction handling.

## What's New

### Tax Compliance
- Add tax profile page for company tax classification (PKP/non-PKP status, NPWP management)

### Transaction Templates
- Implement template versioning - track version numbers and preserve template info on transactions
- Show Debit/Kredit (D/K) account indicators in template selection dropdown on transaction form

## Improvements

- Sort chart of accounts children by account code for consistent ordering

## Bug Fixes

- Add actual balance validation to journal preview - prevent unbalanced entries
- Resolve race condition in transaction form preview causing intermittent display issues
- Display all page numbers in transaction list pagination instead of truncated range

## Breaking Changes

- None

## Migration Guide

No migration steps required. Database schema is unchanged.

## Known Issues

- None

## Dependencies

- Spring Boot 4.0.0
- Java 25
