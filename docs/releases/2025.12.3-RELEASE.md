# Release 2025.12.3

Major release completing multi-industry support with 4 industry seed packs, comprehensive user manuals, and 115 functional tests.

## Highlights

- **Complete Industry Support**: Four production-ready industry seed packs (IT Services, Online Seller, Manufacturing, Education)
- **Comprehensive User Manuals**: 15 documentation files covering all industries with detailed transaction walkthroughs
- **Robust Test Coverage**: 115 functional tests using stable data-testid locators
- **Education Industry**: Full support for universities with student billing, scholarships, and receivables management

## Migration Guide

Standard upgrade path from 2025.12.2-RELEASE:

1. Pull latest code
2. Run `./mvnw clean package`
3. Deploy using standard procedure
4. (Optional) Import industry seed packs via Settings > Import Data

## Industry Seed Pack Import

Users can now import complete industry-specific data:

```bash
# Available in industry-seed/ directory:
- it-service/seed-data/     # IT Services
- online-seller/seed-data/  # Online Seller
- coffee-shop/seed-data/    # Coffee Shop (Manufacturing)
- campus/seed-data/         # University (Education)
```

Import via UI: **Settings > Import Data > Upload ZIP**

## Technical Details

### Test Data Architecture

All functional tests now use:
- Industry-specific seed packs loaded via DataImportService
- TestDataInitializer pattern with `@Profile("functional")`
- Separate test master data in `src/test/resources/testdata/`
- Single source of truth for seed data (no SQL duplication)

### Documentation System

- User manual structured in 12 sections + 4 appendices
- Screenshots generated by functional tests
- Stored in `target/user-manual/screenshots/{industry}/`
- Referenced as `screenshots/{industry}/*.png` in manuals
