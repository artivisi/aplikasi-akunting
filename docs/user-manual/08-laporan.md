# Laporan Keuangan

## Pengertian

Laporan keuangan adalah ringkasan informasi finansial perusahaan yang disusun secara sistematis untuk membantu pengambilan keputusan bisnis. Aplikasi ini menyediakan empat laporan keuangan utama yang umum digunakan.

## Jenis-jenis Laporan

| Laporan | Fungsi |
|---------|--------|
| Neraca Saldo | Memverifikasi keseimbangan debit dan kredit |
| Laporan Laba Rugi | Menunjukkan profitabilitas dalam periode tertentu |
| Neraca | Menampilkan posisi keuangan (aset, liabilitas, ekuitas) |
| Laporan Arus Kas | Melacak pergerakan kas masuk dan keluar |

## Mengakses Laporan

1. Klik menu **Laporan** di sidebar
2. Halaman indeks menampilkan kategori laporan:
   - **Laporan Dasar** - Neraca Saldo, Laba Rugi, Neraca, Arus Kas
   - **Laporan Transaksi** - Daftar Transaksi, Jurnal Umum (segera hadir)
   - **Laporan Analisis** - Analisis Tren, Perbandingan Periode (segera hadir)
3. Klik kartu laporan yang diinginkan

## Neraca Saldo (Trial Balance)

Neraca saldo menampilkan semua akun beserta saldo debit dan kredit untuk memverifikasi keseimbangan pembukuan.

### Cara Menggunakan

1. Klik **Neraca Saldo** di halaman laporan
2. Pilih periode laporan:
   - **Periode** - Pilih bulan dan tahun
3. Klik **Tampilkan** untuk memuat laporan

### Membaca Laporan

Laporan dikelompokkan berdasarkan tipe akun:

| Kelompok | Keterangan |
|----------|------------|
| Aset | Kas, piutang, persediaan, aset tetap |
| Liabilitas | Utang usaha, utang bank |
| Ekuitas | Modal, laba ditahan |
| Pendapatan | Penjualan, pendapatan jasa |
| Beban | Biaya operasional, gaji, utilitas |

### Verifikasi Keseimbangan

Di bagian bawah laporan terdapat:
- **Total Debit** - Jumlah seluruh saldo debit
- **Total Kredit** - Jumlah seluruh saldo kredit
- **Status** - Indikator "Seimbang" jika total debit = kredit

> Catatan: Jika neraca saldo tidak seimbang, periksa kembali jurnal yang telah diposting untuk menemukan kesalahan.

## Laporan Laba Rugi (Income Statement)

Laporan laba rugi menunjukkan pendapatan, beban, dan laba/rugi bersih dalam periode tertentu.

### Cara Menggunakan

1. Klik **Laporan Laba Rugi** di halaman laporan
2. Pilih periode laporan:
   - **Dari Tanggal** - Tanggal awal periode
   - **Sampai Tanggal** - Tanggal akhir periode
3. Klik **Tampilkan**

### Struktur Laporan

```
PENDAPATAN
  Penjualan                     xxx
  Pendapatan Jasa               xxx
  ─────────────────────────────────
  Total Pendapatan                    xxx

BEBAN
  Beban Gaji                    xxx
  Beban Utilitas                xxx
  Beban Operasional             xxx
  ─────────────────────────────────
  Total Beban                         xxx

─────────────────────────────────────────
LABA BERSIH                           xxx
```

### Indikator Keuangan

| Indikator | Rumus | Keterangan |
|-----------|-------|------------|
| Gross Margin | (Pendapatan - HPP) / Pendapatan | Margin kotor sebelum beban operasional |
| Net Margin | Laba Bersih / Pendapatan | Persentase laba dari total pendapatan |

## Neraca (Balance Sheet)

Neraca menampilkan posisi keuangan perusahaan pada tanggal tertentu, terdiri dari Aset, Liabilitas, dan Ekuitas.

### Cara Menggunakan

1. Klik **Neraca** di halaman laporan
2. Pilih tanggal laporan:
   - **Per Tanggal** - Tanggal posisi keuangan yang ingin dilihat
3. Klik **Tampilkan**

### Struktur Laporan

Neraca ditampilkan dalam format dua kolom:

| Kolom Kiri (ASET) | Kolom Kanan (LIABILITAS + EKUITAS) |
|-------------------|-----------------------------------|
| Aset Lancar | Liabilitas Jangka Pendek |
| Aset Tetap | Liabilitas Jangka Panjang |
| | Ekuitas |
| **Total Aset** | **Total Liabilitas + Ekuitas** |

### Persamaan Akuntansi

Neraca harus memenuhi persamaan dasar akuntansi:

```
ASET = LIABILITAS + EKUITAS
```

Jika total di kedua sisi tidak sama, ada kesalahan dalam pencatatan yang perlu diperiksa.

### Komponen Neraca

**Aset Lancar:**
- Kas dan Setara Kas
- Piutang Usaha
- Persediaan
- Biaya Dibayar di Muka

**Aset Tetap:**
- Tanah
- Bangunan
- Peralatan
- Kendaraan
- (Dikurangi Akumulasi Penyusutan)

**Liabilitas Jangka Pendek:**
- Utang Usaha
- Utang Gaji
- Utang Pajak

**Liabilitas Jangka Panjang:**
- Utang Bank
- Utang Obligasi

**Ekuitas:**
- Modal Disetor
- Laba Ditahan

## Laporan Arus Kas (Cash Flow Statement)

Laporan arus kas melacak pergerakan kas masuk dan keluar selama periode tertentu.

### Cara Menggunakan

1. Klik **Laporan Arus Kas** di halaman laporan
2. Pilih periode laporan:
   - **Dari Tanggal** - Tanggal awal periode
   - **Sampai Tanggal** - Tanggal akhir periode
3. Klik **Tampilkan**

### Tiga Kategori Arus Kas

| Kategori | Keterangan | Contoh |
|----------|------------|--------|
| Operasional | Aktivitas bisnis utama | Penerimaan dari pelanggan, pembayaran ke supplier |
| Investasi | Pembelian/penjualan aset | Pembelian peralatan, penjualan kendaraan |
| Pendanaan | Transaksi dengan pemilik/kreditor | Pinjaman bank, pembayaran dividen |

### Struktur Laporan

```
ARUS KAS DARI AKTIVITAS OPERASIONAL
  Penerimaan dari Pelanggan           xxx
  Pembayaran ke Supplier             (xxx)
  Pembayaran Gaji                    (xxx)
  ─────────────────────────────────────────
  Arus Kas Bersih Operasional              xxx

ARUS KAS DARI AKTIVITAS INVESTASI
  Pembelian Aset Tetap               (xxx)
  Penjualan Investasi                 xxx
  ─────────────────────────────────────────
  Arus Kas Bersih Investasi               (xxx)

ARUS KAS DARI AKTIVITAS PENDANAAN
  Penerimaan Pinjaman Bank            xxx
  Pembayaran Utang                   (xxx)
  ─────────────────────────────────────────
  Arus Kas Bersih Pendanaan               xxx

═══════════════════════════════════════════
KENAIKAN/PENURUNAN KAS BERSIH             xxx
Saldo Kas Awal Periode                    xxx
═══════════════════════════════════════════
SALDO KAS AKHIR PERIODE                   xxx
```

## Mencetak dan Mengekspor Laporan

Setiap laporan memiliki tombol aksi di bagian atas:

| Tombol | Fungsi |
|--------|--------|
| Cetak | Membuka dialog print browser |
| Export PDF | Mengunduh laporan dalam format PDF |
| Export Excel | Mengunduh laporan dalam format Excel |

### Tips Pencetakan

1. Gunakan orientasi **Landscape** untuk laporan dengan banyak kolom
2. Aktifkan opsi **Background graphics** untuk mencetak warna
3. Sesuaikan **Scale** jika laporan tidak muat dalam satu halaman

## Tips Menggunakan Laporan

1. **Periksa Neraca Saldo** secara berkala untuk memastikan keseimbangan
2. **Bandingkan Laba Rugi** antar periode untuk melihat tren
3. **Analisis Arus Kas** untuk memastikan likuiditas bisnis
4. **Gunakan filter tanggal** yang konsisten untuk perbandingan yang akurat
5. **Simpan laporan PDF** sebagai arsip bulanan/tahunan
