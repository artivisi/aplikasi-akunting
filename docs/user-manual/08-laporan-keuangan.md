# Laporan Keuangan

## Jenis Laporan

Aplikasi menyediakan tiga laporan keuangan utama:

| Laporan | Fungsi |
|---------|--------|
| **Neraca Saldo** | Daftar saldo semua akun pada tanggal tertentu |
| **Neraca** | Laporan posisi keuangan (aset, kewajiban, ekuitas) |
| **Laba Rugi** | Laporan pendapatan dan beban per periode |

## Neraca Saldo (Trial Balance)

Neraca saldo menampilkan daftar semua akun beserta saldo debit dan kredit pada tanggal tertentu. Digunakan untuk memvalidasi keseimbangan pembukuan.

### Mengakses Neraca Saldo

1. Klik menu **Laporan** di sidebar
2. Pilih **Neraca Saldo**
3. Pilih **Tanggal** laporan
4. Klik **Tampilkan**

### Memahami Neraca Saldo

| Kolom | Keterangan |
|-------|------------|
| Kode Akun | Nomor akun dalam bagan akun |
| Nama Akun | Nama lengkap akun |
| Debit | Total saldo debit akun |
| Kredit | Total saldo kredit akun |

> Total Debit harus sama dengan Total Kredit. Jika berbeda, ada kesalahan pencatatan yang perlu ditelusuri.

## Neraca (Balance Sheet)

Neraca menampilkan posisi keuangan perusahaan pada tanggal tertentu, terdiri dari tiga bagian utama.

### Struktur Neraca

| Bagian | Isi |
|--------|-----|
| **Aset** | Kas, piutang, perlengkapan, peralatan |
| **Kewajiban** | Hutang usaha, hutang pajak |
| **Ekuitas** | Modal, laba ditahan, laba berjalan |

### Mengakses Neraca

1. Klik menu **Laporan** di sidebar
2. Pilih **Neraca**
3. Pilih **Tanggal** laporan
4. Klik **Tampilkan**

### Persamaan Akuntansi

```
Aset = Kewajiban + Ekuitas
```

Neraca yang benar akan selalu memenuhi persamaan ini.

## Laba Rugi (Income Statement)

Laporan laba rugi menampilkan hasil operasi perusahaan selama periode tertentu.

### Struktur Laba Rugi

| Bagian | Isi |
|--------|-----|
| **Pendapatan** | Semua penerimaan dari operasi bisnis |
| **Beban** | Semua pengeluaran untuk operasi bisnis |
| **Laba/Rugi Bersih** | Selisih pendapatan dan beban |

### Mengakses Laba Rugi

1. Klik menu **Laporan** di sidebar
2. Pilih **Laba Rugi**
3. Pilih periode:
   - **Tanggal Awal** - Awal periode laporan
   - **Tanggal Akhir** - Akhir periode laporan
4. Klik **Tampilkan**

## Ekspor Laporan

Semua laporan dapat diekspor dalam dua format:

| Format | Kegunaan |
|--------|----------|
| **PDF** | Untuk cetak dan dokumentasi resmi |
| **Excel** | Untuk analisis lebih lanjut |

### Cara Ekspor

1. Tampilkan laporan yang diinginkan
2. Klik tombol **Ekspor PDF** atau **Ekspor Excel**
3. File akan otomatis terunduh

## Filter dan Periode

### Memilih Periode

- Gunakan date picker untuk memilih tanggal
- Format tanggal: DD/MM/YYYY
- Untuk laporan periodik (Laba Rugi), pilih tanggal awal dan akhir

1. Cetak Neraca Saldo akhir bulan untuk validasi pembukuan
2. Bandingkan Laba Rugi antar periode untuk melihat tren
3. Ekspor ke Excel untuk analisis detail atau presentasi
